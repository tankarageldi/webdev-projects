#1) Merge Strings Alternately

We are given two strings, word1 and word2. Question wants us to merge this two Strings, by adding letters in alternating order, starting from word1. And if a string is longer than the other, we need to append the additional letters onto the end of the merged string. So, since two strings most likely going to be different length, lets say n,m for lengths of word1, word2. For this solution, I used the two pointer technique. We need to use two pointers, one for each string, and iterate through the strings,adding characters one by one to the result string. Here is the algorithm for the solution:

  • initialize i,j = 0
  • initialize an empty string, for appending the result.
  • iterate through the array while pointer i's value is less than word1's length, or pointer j's value is less than word2's length.
    • if the pointer i's value is less than word1's length, means we still have at least one character we didnt go through in the string. append that character to the string, and increase i's value by one.
    • if the pointer j's value is less than word2's length, append the character pointer j is pointing, increase j's value by one.
  • return the result string
By this loop and conditions, we ensure that if any of word1 or word2 has smaller length than the other one, after all of its characters are iterated and appended to the result, rest of the longer string's characters will be appended. And since we're only looping through two arrays length n and m, space complexity will be O(m+n). And space complexity is O(1). 

          
class Solution(object):
    def mergeAlternately(self, word1, word2):
        m = len(word1)
        n = len(word2)
        i = 0
        j = 0
        result = []
        
        while i < m or j < n:
            if i < m:
                result += word1[i]
                i += 1
            if j < n:
                result += word2[j]
                j += 1
        
        return "".join(result)

#2) Greatest Common Divisor of Strings

for this question,For two strings s and t, we say "t divides s" if and only if t is concatenated with itself one or more times. So my approach was, if there is a greatest common divisor, then combining that greatest common divisor 1 or more times with itself will give s, and t. So if there is a greatest common divisor, the result of combining t and s, will be some type of multiplication of the gcd. By this logic, to check if there is a gcd, we can just check if the strings combined from both sides are equal. If they are not, there is no gcd, so we can just return an empty string. If they are equal, we can go on and calculate the gcd of the lengths of the strings, which will give us the largest common divisor length inside the string, lets say m . Now we just have to return the first m letters of any of them word1 or word2. Here is the algorithm :

  • Check if str1 + str2 is not equal to str2 + str1
    • if true, return empty string since there is no greatest common divisor of these two strings.
  • Else, calculate the greatest commmon divisor between str1 and str2's lengths, assign it to a variable "n".
  • return the first n characters in any string. 

          
class Solution:
    def gcdOfStrings(self, str1: str, str2: str)
        if str1 + str2 != str2 + str1:
            return ""
                
        max_length = gcd(len(str1),len(str2))
        return str1[:max_length]

#6) Reverse Words in String

For this question, my first intuition was converting the string into a list by splitting the words by empty spaces after them. Python has a function for this, which is split() function. After the spliting the string to a list, we are going to iterate the list in reverse order. And while iterating in reverse order, we need to make sure that spaces between words are correctly appended too. For this, we will not append the first word in the original list. Here is the algorithm :

  • split the string into a list.
  • create a variable, for appending words.
  • iterate through the list, in reverse order, but without the first element of the original list.
    • append the first element in reverse list to the result string, also append a " " character.
  • get the last word to append to the list, which is the first element of the initial list. append it to the result
  • return the result

          
class Solution:
    def reverseWords(self, s: str) -> str:
        res = ""
        list_words = s.split()
        for item in reversed(list_words[1:]):
            res += item + " "
        last = list_words[0]
        res += last
        return res

#7) Product of Array Except Self

For this question, my first approach was to calculate the product of all elements, and divide each while iterating through the array, but in the description division is not allowed to use. Also they want us to do this question in O(n) time, with O(1) space, not counting the output array. So, second approach was to calculate the left product of each element, and calculate the right product of each element. To do this, we can create two lists, left and right. left list is for keeping track of the left products of each element in nums, and right list is for keeping track of right products of each element in nums, and the answer will be the product of left and right. This way, we find an easy solution with O(n) time, but space complexity is greater than O(1), since we need to have three arrays, left,right,and output. To reduce the complexity, instead of having the right list, we can only calculate the left list, and multiply the corresponding right product value, which we can keep in an integer. And this left list, by this method, turns into the output list, so we dont need that one anymore either. Here is the algorithm for this solution:

  • get the lenght of nums, store it in "n"
  • create the output list, same length as nums list
  • append 1, to the first index of the list. Regardless what element is in the first index, 1 will always be its left product.
  • create a variable, called right, and assign 1, for keeping track of the right product of each index
  • loop from index i = 1 to the end of the nums
      calculate the product of nums[i-1] and output[i-1], and assign it to output[i]
  • loop from index i = length - 1 to 0
    • multiply nums[i] and r values, and assign it to r
    • calculate the product of output[i-1] and r,assign it to output[i-1]
  • return output
O(n) time complexity, O(1) space complexity, when output array space does not count. 

            
class Solution:
    def productExceptSelf(self, nums: list[int]) -> list[int]:
        length = len(nums)
        output = [None] * (length-1)
        output.insert(0,1)
        r = 1

        for i in range(1,length):
            output[i] = nums[i-1] * output[i-1]

        for i in range(length-1,0,-1):
            r = nums[i] * r
            output[i-1] = output[i-1] * r
            
        return output

#8) Increasing Triplet Subsequence

Given an integer array nums, return true if there exists a triple of indices (i, j, k) such that i < j < k and nums[i] < nums[j] < nums[k]. If no such indices exists, return false. First off, this looks like an easy question, but one part is very important, which is the question wants any (i,j,k) indices that is increasing,not consequent i,j,k's only So I struggled a little bit trying to do it with sliding window algorithm, then i read couple solution approaches. Here is the approach i liked the most, In this approach, we are going to keep track of first, and second smallest numbers during the iteration. To be the second smallest number, the number has to be larger than the first one, and this way, we only need to then find a number that is larger than both of them to return true. And here is one case, where the output is true, but the resulting array is not the same as questions resulting array,
nums = [20,100,10,12,5,13]
output = True
results = [5,12,13] # first, second, largest
but it should've been [10,12,13]. If we look at the literal sense, our solution is wrong, but technically, while we are checking the second smallest element, we are making sure that there is an element exists before the second smallest element, that is smaller than second smallest element. So when 5 comes after 12, and the algorithm assigns it to the variable first, it really doesnt matter since there is another element in the array before the second smallest elements index, that will satisfy the condition. So here is the algorihtm:

  • assign first and second variables to the largest integers possible.
  • iterate through the array
    • if the current element is smaller or equal to variable first, assign that element to first.
    • if the current element is smaller than variable second, and larger than first, assign that element to second.
    • if the current element is greater than both of second and first values, then return true.
  • return false if the loop did not return true. 

              
class Solution:
    def increasingTriplet(self, nums: list[int]) -> bool:
        size = len(nums)
        ans = False
        first = float("inf")
        second = float("inf")

        for i in range(size):
            if nums[i] <= first:
                first = nums[i]
            elif nums[i] <= second and nums[i] > first:
                second = nums[i]
            elif nums[i] > first and nums[i] > second:
                ans = True
        return ans

#12) Container With Most Water

You are given an integer array height of length n. There are n vertical lines drawn such that the two endpoints of the ith line are (i, 0) and (i, height[i]). Find two lines that together with the x-axis form a container, such that the container contains the most water. Return the maximum amount of water a container can store. Notice that you may not slant the container.

For this question, we need to find the maximum area, from one element to another in the list. My first thought was, using two pointers, at the opposite sides of the list, and calculate the area of the container those two elements make. To do this, we need to get the difference between the indexes of two elements, and since its a container, its only going to fill up to smallest elements height.So we need to multiply the smallest element out of two we are currently pointing, with the difference of indexes, and we get the current area.If we do this with all the elements in the table by shifting the smallest current element pointer only, we will have a fast solution.
Here is the algorithm:

  • initialize two pointers, left and right, left starting from first element, right starting from last element in the list.
  • initialize max_area variable, set it to 0, we will use this to compare the current area and the max area.
  • iterate through the list while left is smaller than right.
    • do the calculation i mentioned above, multiply the smallest element of two current elements, with difference of two indexes.
    • if this current area is greater than the max_area , then assign it to max_area.
    • to shift the pointers, check the two elements we're pointing at, and shift the smaller elements index by one.
  • now the loop is over, we found the max area. return max_area 

                
class Solution:
    def maxArea(self, height: list[int]) -> int:
        i = 0
        j = len(height) - 1
        max_area = 0
        while i < j:
            max_area = max(min(height[j],height[i]) * (j-i),max_area)
            if height[i] > height[j]:
                j -= 1
            else:
                i += 1

        return max_area

#20) Difference of Two Arrays

Given two 0-indexed integer arrays nums1 and nums2, return a list answer of size 2 where: answer[0] is a list of all distinct integers in nums1 which are not present in nums2. answer[1] is a list of all distinct integers in nums2 which are not present in nums1. Note that the integers in the lists may be returned in any order.

This is a easy question, if we convert both lists to sets, we will get rid of the duplicate elements in each list, and all we need to do after that is get the difference of two sets, from both sides, and we have our answer.

Here is the algorithm:

  • convert nums1,nums2 to sets
  • get the difference of sets from both sides, meaning s1-s2 and s2-s1
    • convert this two differences into lists, and return as a pair 

                
class Solution:
    def findDifference(self, nums1: list[int], nums2: list[int]) -> list[list[int]]:
        s1 = set(nums1)
        s2 = set(nums2)
        a = list(s1.difference(s2))
        b = list(s2.difference(s1))
        return a,b

07/18/2024) 1380. Lucky Numbers in Matrix

Problem : Given an m x n matrix of distinct numbers, return all lucky numbers in the matrix in any order. A lucky number is an element of the matrix such that it is the minimum element in its row and maximum in its column.

This is a matrix question. Since the question explains the lucky number is the minimum element in its row and maximum element in its column. So my first thought was to find the minimum element in each row and put it inside a list. after this is done, we need to find the maximum element in each column, but columns are not as easy to access as rows.To get the max of each column, I constructed a nested for loop, to go through the every i'th element of each row, and append those elements into a dummy list, to basically construct a column list. After first column is constructed inside the list, I calculate the maximum value inside the dummy list, and append this value to another list for keeping track of maximum values in each column, then clear the dummy list to use it again for the next column. After our nested for loop is done, we have two lists, one contains the minimum elements in each row, and other one contains the maximum elements in each column. now go through both of the lists again to find all common numbers, which are going to be our answers. I appended these numbers into the dummy list i used earlier because it was empty,since i clear the array after each use. Here is the algorithm :

  • create three lists, min_row for collecting the minimum value in each row, max_col for collecting the maximum value in each column, dummy for a helper list.
  • for each row in the matrix, append the minimum value in the row to the min_row list.
  • loop from i = 0 until m, where m is length of each row.
    • loop from j = 0 until n, where n is length of each column
      • append the j'th rows i'th column to the dummy list
    • append the maximum value in the dummy list to the max_col list.
    • clear the dummy list
  • go through each element1 in min_row
    • go through each element2 in max_col
    • if element1 is equal to element 2, append the value to the dummy list
  • return dummy list

                
class Solution:
    def luckyNumbers (self, matrix: list[list[int]]) -> list[int]:
        n = len(matrix)
        m = len(matrix[0])
        min_row = []
        max_col = []
        checklist = []
        for row in matrix:
            min_row.append(min(row))
        for i in range(m):
            for j in range(n):
                checklist.append(matrix[j][i])
            max_col.append(max(checklist))
            checklist.clear()
        for element1 in min_row:
            for element2 in max_col:
                if element1 == element2:
                    checklist.append(element1)
        return checklist

Description: You are given a 0-indexed integer array mapping which represents the mapping rule of a shuffled decimal system. mapping[i] = j means digit i should be mapped to digit j in this system.

Notes: Elements with the same mapped values should appear in the same relative order as in the input. The elements of nums should only be sorted based on their mapped values and not be replaced by them.

Here is the algorithm:

  • create an array to hold pairs of the converted number, and the index of the corresponding element in the nums array.
  • iterate through the array nums from i = 0 to the end.
    • convert the current element in the iteration to a string, to operate with its index

      • get the element from mapping array with this index. convert it to string and append it to the element string.
    • after the nested iteration is done, convert the element to integer.
    • append the value and i pairs to the array that we created to hold the pairs.
  • sort the pairs array.
  • create an array for final result
  • loop through each pair in pairs
    • get the i from the pair, get nums[i], and append it to the result array.
  • return the result array

                
class Solution:
    def sortJumbled(self, mapping: list[int], nums: list[int]) -> list[int]:
        pairs = []
        n = len(nums)
        for i in range(n):
                s = str(nums[i])
                element = ""

07/25/2024) 912. Sort an Array

Given an array of integers nums, sort the array in ascending order and return it. You must solve the problem without using any built-in functions in O(nlog(n)) time complexity and with the smallest space complexity possible. Since we can't use built in functions, and need to solve it in O(nlog(n)) time, best two options were heap sort, or merge sort. I implemented a heap sort algorithm, which has O(nlogn) complexity, both best and worst case.

Here is the link to heap sort algorithm, using heapifying array and tree structure to sort the array. link 

                
class Solution:
    def heapify(self,arr,n,i):
        largest = i
        l = 2 * i + 1
        r = 2 * i + 2

        if l < n and arr[i] < arr[l]:
            largest = l 
        
        if r < n and arr[largest] < arr[r]:
            largest = r
        
        if largest != i:
            arr[i],arr[largest] = arr[largest], arr[i]
            self.heapify(arr,n,largest)

    def sortArray(self, nums: list[int]) -> list[int]:
    
        n = len(nums)
        for i in range(n//2 , -1 , -1):
            self.heapify(nums,n,i)
        for i in range(n-1,0,-1):
            nums[i],nums[0] = nums[0],nums[i]
            self.heapify(nums,i,0)
        return nums

07/30/2024) 1653. Minimum Deletions to make a String Balanced

Problem Explanation: You are given a string s consisting only of characters 'a' and 'b'​​​​. You can delete any number of characters in s to make s balanced. s is balanced if there is no pair of indices (i,j) such that i < j and s[i] = 'b' and s[j]= 'a'. Return the minimum number of deletions needed to make s balanced.

You can solve this question with multiple approaches, but the one i liked the most is the stack approach. We need to find all 'ba' pairs where b comes before a, and keep the count for every pair we found. This count is going to be our final answer for minimum deletions. Algorithm :

  • initialize a stack, and a counter.
  • for each character in the string;
    • if the stack is not empty, and the top character of the stack is b, it means if the current character is a, we found a 'ba' pair.
    • pop the top character, and increase the counter by one
    • else just append the current character to the stack
  • return the counter 

                
  class Solution:
      def minimumDeletions(self, s: str) -> int:
          stack = []
          delete = 0

          for char in s:
              if stack and stack[-1] == "b" and char == "a":
                  stack.pop()
                  delete += 1
              else:
                  stack.append(char)
          return delete

334. Reverse String

Write a function that reverses a string. We have an char array representing a string, and we need to reverse this. So i have approached this question with two pointers, on the opposide sides of the char array. We need to iterate throught the list until left pointer reaches right pointer, and swap the values. Here's the algorithm:

  • initialize two pointers, left = 0, right = far right element in the list.
  • iterate through the array until left pointer value is less than right pointer value
  • swap the left indexed element with right indexed element,
  • increase left pointer by one, decrease right pointer by one.

Since we dont need to return anything, problem is solved. 

              
class Solution:
    def reverseString(self, s: list[str]) -> None:
        left = 0
        right = len(s) - 1
        while left < right:
            s[left],s[right] = s[right],s[left]
            left += 1
            right -= 1

342. Power of Four

Given an integer n, return true if it is a power of four. Otherwise, return false. An integer n is a power of four, if there exists an integer x such that n == 4x.
So, for solving this question, we just need to find the value x, such as n == 4^x. For this, we can take natural logs of both sides, which will give us ln(n) == x ln(4), and if we rearrange the elements, we get x = ln(n)/ln(4). So we calculate this value of x, since we know n. If this value of x is an integer value, it means that N is power of four. If not, then it is not. Numbers less and equal to 0 are exceptions, because we can take logs of them, they are always going to be false.
Here is the algorithm:

  • if n is less or equal to 0, return false
  • calculate the ln(n)/ln(4), assign it to x
  • return true if x is an integer, false if not. 

            
import math
class Solution:
    def isPowerOfFour(self, n: int) -> bool:
        if n <= 0:
            return False
        x = math.log(n)/math.log(4)
        return x.is_integer()

217.Contains Duplicate

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.
For this question, i Used a dictionary to add the values, and while adding the valuesm if a duplicate value is added to the dictionary, it will return true and stop the program. here is the algorithm :

  • initialize an hash map
  • for each number in nums, if the number is in the hash map as a key, return true, if not, add it to the map.
  • return false if the program reaches this line. 

            
class Solution:
    def containsDuplicate(self, nums: list[int]) -> bool:
        dic = {}
        for num in nums:
            if num in dic:
                return True
            else:
                dic[num] = 1
        return False
